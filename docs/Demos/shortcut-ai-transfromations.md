# Demo 3 - Shortcut AI Transformations

This demo showcases how easy it is to create transformed shortcuts in OneLake using built-in AI Capabilities. This will on showcasing PII redaction from txt files and how dynamic it is just like the shortcut file transformations. By uploading new txt files to the OneLake shortcut, the PII redaction will be applied automatically, ensuring that sensitive information is protected without any manual intervention.

## Pre-Requisites

1. An Azure Storage Account provisioned on Azure with the following container:

- `support-chats-export`

1. Upload all the txt files from the [`data/support_chats/az-storage`](../data/support_chats/az-storage) folder to the `support-chats-export` container.
