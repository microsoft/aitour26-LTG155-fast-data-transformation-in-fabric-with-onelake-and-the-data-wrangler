# Fast data transformation in Fabric with OneLake and the Data Wrangler

## Session Abstract

Prepping data shouldn’t feel like a chore. See how multi-cloud OneLake shortcuts connect to data sources without duplication, run shortcut file transforms and shortcut AI transforms like summarisation and PII detection, and use Data Wrangler to clean and shape results, inside Microsoft Fabric.
